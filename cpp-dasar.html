Apa itu C++?

C++ adalah bahasa pemrograman yang dikembangkan sebagai ekstensi dari bahasa C. C++ digunakan untuk membuat berbagai aplikasi, termasuk perangkat lunak sistem, game, dan aplikasi desktop. Bahasa ini mendukung pemrograman berorientasi objek dan memiliki sintaks yang mirip dengan bahasa C.

Program Pertama: Hello World

Program "Hello World" adalah program pertama yang biasanya dibuat oleh pemula untuk memastikan bahwa lingkungan pengembangan telah disiapkan dengan benar. Berikut adalah contoh kode untuk program "Hello World" dalam C++:


#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Setelah menulis kode di atas, kompilasi dan jalankan program. Jika berhasil, pesan "Hello, World!" akan ditampilkan di layar.

Variabel dan Tipe Data

Variabel digunakan untuk menyimpan data. C++ memiliki berbagai tipe data, seperti:

  • int: Bilangan bulat (contoh: int angka = 10;)
  • float: Bilangan desimal (contoh: float harga = 19.99;)
  • char: Karakter tunggal (contoh: char huruf = 'A';)
  • string: Teks (contoh: string nama = "Budi";)

Contoh penggunaan variabel:


#include <iostream>
using namespace std;

int main() {
    int umur = 20;
    string nama = "Budi";
    cout << "Nama: " << nama << ", Umur: " << umur << endl;
    return 0;
}

Operator Aritmatika

C++ mendukung berbagai operator aritmatika, seperti:

  • +: Penjumlahan
  • -: Pengurangan
  • *: Perkalian
  • /: Pembagian
  • %: Modulus (sisa bagi)

Contoh penggunaan operator aritmatika:


#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int b = 5;
    cout << "Penjumlahan: " << a + b << endl;
    cout << "Pengurangan: " << a - b << endl;
    cout << "Perkalian: " << a * b << endl;
    cout << "Pembagian: " << a / b << endl;
    cout << "Modulus: " << a % b << endl;
    return 0;
}

Pernyataan Kondisional

Untuk membuat keputusan dalam program, C++ menyediakan pernyataan kondisional seperti if, else if, dan else.

Contoh penggunaan pernyataan kondisional:


#include <iostream>
using namespace std;

int main() {
    int nilai = 85;
    if (nilai >= 75) {
        cout << "Lulus!" << endl;
    } else {
        cout << "Tidak Lulus." << endl;
    }
    return 0;
}

Perulangan

C++ mendukung berbagai jenis perulangan, seperti for, while, dan do-while.

Contoh penggunaan perulangan for:


#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << "Iterasi ke-" << i << endl;
    }
    return 0;
}
