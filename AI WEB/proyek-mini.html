Proyek: Menyalakan LED dengan Tombol

Dalam proyek ini, kita akan membuat sistem yang menyalakan LED ketika tombol ditekan. Ini adalah proyek sederhana yang mengajarkan tentang penggunaan input (tombol) dan output (LED) pada Arduino.

Komponen yang Diperlukan

  • Arduino Uno
  • LED
  • Resistor 220Ω
  • Tombol
  • Jumper wires
  • Breadboard

Hubungkan komponen-komponen seperti pada diagram berikut:

Diagram Rangkaian

Hubungkan LED pada pin 13 dan tombol pada pin 2, dengan menggunakan resistor sesuai dengan instruksi berikut:

  • Pin 13 -> LED -> Resistor 220Ω -> Ground
  • Pin 2 -> Tombol -> Ground (dengan pull-up resistor internal)

Kode Program

Berikut adalah kode untuk menyalakan LED ketika tombol ditekan:


int ledPin = 13;      // LED di pin 13
int buttonPin = 2;    // Tombol di pin 2
int buttonState = 0;   // Variabel untuk membaca status tombol

void setup() {
  pinMode(ledPin, OUTPUT);  // Set pin LED sebagai output
  pinMode(buttonPin, INPUT);  // Set pin tombol sebagai input
}

void loop() {
  buttonState = digitalRead(buttonPin);  // Baca status tombol

  if (buttonState == HIGH) {  // Jika tombol ditekan
    digitalWrite(ledPin, HIGH);  // Nyalakan LED
  } else {
    digitalWrite(ledPin, LOW);  // Matikan LED
  }
}

Upload kode ini ke Arduino kamu dan tombol akan mengontrol LED di pin 13.
